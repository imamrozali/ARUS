# @arus/common

Shared utilities, helpers, constants, and types.

## Installation

```bash
npm install @arus/common
```

## API Reference

### Helpers
- `normalizePath(path: string): string`
- `isAsyncFunction(fn: unknown): boolean`
- `safeStringify(obj: unknown): string`

### Constants
- `SUPPORTED_METHODS`
- `DEFAULT_HEADERS`

### Types
- `HttpMethod`
- `Headers`
- `JsonSerializable`

See [main README](../../README.md) for usage examples.
